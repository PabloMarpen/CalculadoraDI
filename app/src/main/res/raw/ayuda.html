Ayuda para el uso de la Calculadora

Bienvenido a la sección de ayuda de la calculadora. Aquí te explicamos cómo utilizarla.

Funciones Básicas

  • Suma (+): Utiliza este botón para sumar dos números.
  • Resta (-): Usa este botón para restar un número de otro.
  • Multiplicación (×): Este botón multiplica dos números.
  • División (÷): Utiliza este botón para dividir un número entre otro.

Ejemplo de Uso

Para sumar 5 y 3:

  1. Introduce 5.
  2. Pulsa el botón +.
  3. Introduce 3.
  4. Pulsa el botón = para ver el resultado.

Consejos

  • Asegúrate de pulsar el botón AC o CA para limpiar la pantalla antes de comenzar una nueva operación.
